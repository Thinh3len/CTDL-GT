# 📚 Cấu trúc Dữ liệu và Giải thuật (CTDL&GT)

Chào mừng bạn đến với kho lưu trữ code môn học Cấu trúc Dữ liệu & Giải thuật của mình! 🚀 
Kho này dùng để lưu trữ các bài tập và thuật toán kinh điển được cài đặt bằng ngôn ngữ **C++**.

## 📂 Nội dung Repository

Hiện tại, repository bao gồm các thư mục và file sau:

### Chương 2: Tìm kiếm & Sắp xếp (`Chuong2_TimKiem_SapXep.cpp`)
File này chứa các thuật toán tìm kiếm và sắp xếp cơ bản cùng với hàm test chạy thử (`DemoSapXep`, `DemoTimKiem`).

**🔍 Các thuật toán Tìm kiếm:**
- Tìm kiếm tuyến tính (Linear Search) - `O(N)`
- Tìm kiếm nhị phân (Binary Search) - `O(log N)`

**🔄 Các thuật toán Sắp xếp:**
- Chọn trực tiếp (Selection Sort) - `O(N^2)`
- Chèn trực tiếp (Insertion Sort) - `O(N^2)`
- Nổi bọt (Bubble Sort) - `O(N^2)`
- Sắp xếp Nhanh (Quick Sort) - `O(N log N)`
- Vun đống (Heap Sort) - `O(N log N)`
- ✨ **Đặc biệt:** Có bổ sung thuật toán Chèn nhị phân (Binary Insertion Sort).

## 🛠️ Hướng dẫn sử dụng

2. Mở bằng bất kỳ IDE C++ nào (Dev-C++, Visual Studio, Code::Blocks,...).
3. Biên dịch và chạy file để xem kết quả xuất ra màn hình console.
